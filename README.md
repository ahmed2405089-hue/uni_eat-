
 Overview

UniEats is a web-based platform that allows university students to browse campus restaurants, place food orders, and track their order status.

The system is designed to simplify the ordering process inside the university by reducing waiting time and organizing food pickup efficiently.

## Features

- User registration and login
- Browse restaurants and menus
- Add items to cart
- Place orders
- Track order status
- Admin dashboard for managing users, restaurants, and orders
- Owner dashboard for managing menus and orders

## How to Run

1. Open `index.html` in your web browser.
2. Register a new account or login.
3. Select your role: Student, Owner, or Admin.
4. Explore the features based on your role.

## Project Structure

- `pages/`: HTML pages
- `styles/`: CSS stylesheets
- `script/`: JavaScript files
- `assets/`: Images and assets
- `icons/`: Icons

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage for data persistence
